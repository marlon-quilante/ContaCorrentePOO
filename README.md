# Controle de Conta Corrente

![](https://imgur.com/tyjHMp5.gif)

## Introdução

O programa consiste no controle de conta corrente, onde é possível criar várias contas e realizar operações sobre elas e entre elas.

## Funcionalidades

- **Criar Conta:** O sistema permite a criação de várias contas diferentes, que recebem como dados iniciais um número, um saldo e um limite de débito.
- **Realizar Saques:** O sistema permite o saque de valores de uma determinada conta, levando como parâmetro o número dessa conta. O saque só é permitido caso o valor seja menor ou igual a soma do saldo da conta e o limite de débito.
- **Realizar Depósitos:** O sistema permite o depósito de valores de uma determinada conta, levando como parâmetro o número dessa conta.
- **Consulta de Saldo:** O sistema permite que seja realizado a consulta de saldo de uma determinada conta, levando como parâmetro o número dessa conta
- **Transferência Entre Contas:** O sistema permite que sejam realizadas transferências de valores entre duas contas, levando como parâmetro o número da conta de origem e o número da conta de destino. A transferência só é permitida caso o valor seja menor ou igual ao saldo da conta.
- **Emissão de Extrato:** O sistema permite gerar um histórico de movimentações de uma determinada conta, levando como parâmetro o número dessa conta.
